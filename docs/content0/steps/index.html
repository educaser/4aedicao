O QUE PRECISO SABER E FAZER PARA ACOMPANHAR OS MINICURSOS?

1. O QUE É R?

R é um software livre voltado principalmente para tratamento e análise de dados. De maneira mais estendida, pode ser visto como uma linguagem de programação especializada na gestão e análise de dados. Disponível para os sistemas operacionais Windows, Linux e MacOS.

POSSO USAR O R SEM A NECESSIDADE DE APRENDER PROGRAMAÇÃO EM R?

Apesar de possuir uma interface padrão pouco amigável apta a receber apenas linhas de comando, o software R possui uma gama de pacotes. Alguns destes pacotes fornecem interfaces bastante amigáveis por meio das quais não é necessário aprender linguagem de programação R para fazer análises de dados e visualizações gráficas, tal como o pacote Rcmdr, que será o ambiente utilizado no Minicurso 1.

2. COMO BAIXAR E INSTALAR O R?

O download do R pode ser feito em https://www.r-project.org . Uma vez nesta página, clique em CRAN e, na sequência, em um dos mirrors disponíveis. Clique agora na opção de download conforme seu sistema operacional. No caso do Windows, clica em seguida em “base” e, por fim, no link em destaque (algo do tipo “Download R-4.3.0 for Windows”). Alternativamente, você pode baixar o R clicando neste link: https://cran.fiocruz.br/bin/windows/base/R-4.3.0-win.exe.

3. BAIXANDO E INSTALANDO O RSTUDIO

Para os Minicursos 2 e 3, será necessário o uso do IDE (Integrated Development Environment, ou ambiente integrado de desenvolvimento, em tradução livre) Rstudio, que deve ser instalado após a instalação do software R (o Rstudio não funciona sem o R). Seu download está disponível no link https://posit.co/download/rstudio-desktop/#download, seja para Windows, Linux ou MacOs. Apesar da utilização do Rstudio no lugar da “interface padrão” do R ainda necessitar substancialmente do uso de linhas de comando, o Rstudio facilita o preenchimento destas e também fornece outras facilidades que tornam menos árdua a geração e compreensão de linhas de comando, sendo massivamente utilizado pelos usuários de R.

4. BAIXANDO E INSTALANDO O VS CODE

Além do Rstudio, o Minicurso 3 usará o Visual Studio Code (VS Code), que é um editor de código, de código aberto, desenvolvido pela Microsoft e disponível para Windows, Linux e MacOS. Seu download está disponível em https://code.visualstudio.com/.
